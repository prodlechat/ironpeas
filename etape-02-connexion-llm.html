1
Comprendre les coûts
Combien ça coûte, et comment éviter les mauvaises surprises
💡 Pourquoi ?

OpenClaw a besoin d'un "cerveau" (un modèle d'IA comme Claude ou GPT) pour fonctionner. Ce cerveau coûte de l'argent à chaque utilisation — on appelle ça les tokens. Avant de continuer, il faut comprendre combien ça coûte pour ne pas avoir de mauvaise surprise.

Le coût total par mois :

Poste Coût Comparable à...
VPS Hostinger (KVM2) ~9,59 €/mois Un abonnement Netflix
Tokens IA — Haiku ~3-10 €/mois Un ou deux cafés
Ubuntu, OpenClaw, Telegram Gratuit —
Total ~13-20 €/mois Un resto rapide

Les modèles disponibles :

Modèle Prix Pour quoi faire
Claude Haiku Très peu cher Trier emails, résumés, agenda — parfait pour débuter
Claude Sonnet Moyen Rédaction, analyse — quand Haiku ne suffit pas
Claude Opus Cher Tâches complexes — à éviter au quotidien
Kimi 2.5 Très peu cher Alternative à Haiku, clé API gratuite au départ
💰
On va utiliser Haiku pour cette installation. C'est le modèle le plus économique et il suffit pour 90% des usages courants. Vous pourrez changer à tout moment.
💡
Conseil : Avec 10$ de crédits Anthropic et le modèle Haiku, vous pouvez utiliser OpenClaw pendant des semaines en usage normal (tri emails, résumés, gestion agenda). Pas besoin de mettre 50$ d'entrée de jeu.
⚠️
Note sur le mode OAuth : Si vous avez déjà un abonnement Claude Pro/Max, vous pourriez être tenté de l'utiliser directement avec OpenClaw au lieu de payer des tokens. C'est techniquement possible, mais Anthropic considère cela comme une violation de ses conditions d'utilisation et peut bannir votre compte. On recommande la clé API pour être tranquille.
2
Créer sa clé API Anthropic
Le badge d'accès qui permet à OpenClaw d'utiliser le cerveau Claude
💡 Pourquoi ?

La clé API, c'est le badge d'accès qui permet à OpenClaw d'utiliser le cerveau Claude. Sans clé, pas de cerveau, pas de réponses.

Voici les étapes :

  1. Allez sur console.anthropic.com (copiez le lien ci-dessous et collez-le dans votre navigateur)
  2. Créez un compte si vous n'en avez pas (email + mot de passe classique)
  3. Une fois connecté, allez dans API Keys (dans le menu de gauche)
  4. Cliquez sur Create Key (Créer une clé)
  5. Donnez-lui un nom (par exemple openclaw)
  6. COPIEZ LA CLÉ IMMÉDIATEMENT — elle ne sera plus jamais affichée !
  7. Collez-la dans un endroit sûr (Notes, document, 1Password...)
  8. Allez dans Billing (Facturation) et rechargez avec 10$ minimum
Adresse web — collez dans votre navigateur, PAS dans le terminal 
https://console.anthropic.com
✅ Si vous voyez votre clé dans la liste des API Keys, et que votre solde affiche au moins 10$, c'est bon !
🔴
Attention : La clé API ne s'affiche qu'UNE SEULE FOIS au moment de la création. Si vous fermez la fenêtre sans la copier, il faudra en créer une nouvelle. Copiez-la AVANT de faire quoi que ce soit d'autre.

Problème ?

▶ "J'ai fermé la fenêtre sans copier la clé"
Pas de panique. Retournez dans API Keys, supprimez l'ancienne clé, et créez-en une nouvelle. Pensez à la copier cette fois !
▶ "Je ne trouve pas le menu Billing"
Dans le menu de gauche de la console Anthropic, cherchez Settings puis Billing. Si vous ne le trouvez pas, essayez l'URL directe : console.anthropic.com/settings/billing
▶ "Ma carte est refusée"
Anthropic n'accepte que certaines cartes bancaires. Essayez avec une autre carte ou une carte virtuelle (Revolut, N26, etc.). Les cartes prépayées sont parfois refusées.
3
Installer OpenClaw sur le VPS
C'est LE moment — votre agent IA va exister
💡 Pourquoi ?

C'est LE moment : on installe OpenClaw sur votre VPS. Après ça, votre agent IA existera.

🔴
IMPORTANT — Où taper cette commande :
Cette commande doit être tapée dans votre terminal SSH — celui où vous voyez openclaw@srv-quelquechose au début de la ligne.
  • PAS dans le terminal Hostinger (dans le navigateur)
  • PAS directement sur votre Mac/PC (sinon OpenClaw s'installe sur VOTRE ordinateur et a accès à tout !)
🔍 Comment vérifier où vous êtes ?

Regardez le début de la ligne de commande :

  • openclaw@srv... → Vous êtes au bon endroit
  • votre-nom@MacBook... ou C:\Users\... → Vous êtes sur votre ordi ! Reconnectez-vous d'abord en SSH (Étape N°1)

Copiez la commande ci-dessous et collez-la dans le terminal SSH :

Terminal SSH — en tant qu'openclaw 
curl -fsSL --proto '=https' --tlsv1.2 https://openclaw.ai/install.sh | bash
⚠️
Vérifiez DEUX FOIS que vous êtes bien sur le serveur (prompt openclaw@srv...) avant de lancer cette commande. Si vous l'installez par erreur sur votre Mac/PC, OpenClaw aura accès à TOUT votre ordinateur personnel — vos photos, vos fichiers, vos mots de passe.
✅ Si l'installation se termine sans erreur rouge et que vous voyez un message de succès ou un menu d'onboarding, c'est parfait !

Problème ?

▶ "curl: command not found"
Curl n'est pas installé. Tapez d'abord : sudo apt-get install -y curl puis relancez la commande d'installation.
▶ "Permission denied" pendant l'installation
Vous n'êtes peut-être pas connecté en tant que openclaw. Vérifiez le prompt. Si nécessaire : su - openclaw
▶ L'installation ne démarre pas ou plante
Vérifiez votre connexion internet sur le VPS : ping -c 3 google.com. Si ça ne répond pas, redémarrez le VPS depuis Hostinger.
▶ "brew: command not found" après l'installation
Homebrew n'est pas dans le PATH. Tapez : eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)" puis relancez.
4
Configurer OpenClaw (onboarding)
La fiche d'arrivée de votre nouvel employé IA
💡 Pourquoi ?

OpenClaw vous pose des questions pour se configurer : quel cerveau utiliser, quelles compétences activer. C'est comme remplir la fiche d'arrivée d'un nouvel employé.

L'installateur va vous poser plusieurs questions. Voici exactement quoi répondre :

  1. Quickstart ou Manual ? → Choisissez Quickstart
  2. "I understand this is personal..." → Répondez Yes (vous confirmez que c'est pour votre usage)
  3. Auth method (méthode de connexion au cerveau IA) → Choisissez Anthropic API key
  4. "How do you want to provide this API key?" → Choisissez Environment variable (c'est la méthode la plus simple)
  5. Collez votre clé API (celle de la sous-étape 2 — copiez-collez-la depuis vos notes)
  6. Modèle → Choisissez Haiku (pas Opus, beaucoup trop cher pour débuter !)
⏸️
L'onboarding va ensuite vous demander votre token Telegram. Pas de panique — on le crée ensemble à l'Étape N°3. La suite au prochain module !
💡
Conseil : Ne fermez pas le terminal ! L'onboarding est en pause, il attend votre réponse. Passez à l'Étape N°3 pour créer votre bot Telegram et obtenir le token, puis revenez ici pour continuer.

Problème ?

▶ "J'ai fermé le terminal par erreur"
Pas de panique. Reconnectez-vous en SSH (ssh -L 6080:localhost:6080 openclaw@VOTRE_IP) et relancez l'onboarding. Vos paramètres précédents sont souvent sauvegardés.
▶ "La clé API est refusée / Invalid API key"
Vérifiez que vous avez bien copié la clé en entier (elle commence par sk-ant-). Assurez-vous aussi que votre solde Billing est bien rechargé (10$ minimum).
▶ "Je ne vois pas l'option Haiku"
Les noms de modèles peuvent varier selon la version d'OpenClaw. Cherchez le modèle le moins cher / le plus rapide dans la liste proposée.
▶ "J'ai choisi Opus par erreur"
Vous pourrez changer le modèle dans la configuration d'OpenClaw après l'installation. Cherchez le fichier de config ou relancez l'onboarding.
▶ "L'onboarding est bloqué / ne répond plus"
Appuyez sur Ctrl+C pour arrêter, puis relancez la commande d'installation. Vos précédentes réponses sont généralement sauvegardées.
